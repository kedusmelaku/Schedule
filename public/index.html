Google Calendar Access Needed

Scheduler uses Google Calendar to read student bookings and automatically fill in student counts when generating a schedule.

To grant access:

  1. Stop the server
  2. Delete token.json from the project folder
  3. Restart with node server.js and follow the URL it prints

If you'd prefer not to connect Google Calendar, you can continue and enter student counts manually.
